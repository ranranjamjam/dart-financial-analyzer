# DART 사업보고서 재무 분석기 설계 문서

## 목적

투자심사역 직무 전환을 준비하며, DART 사업보고서 PDF를 기반으로 기업의 재무 상태를 분석하고 학습할 수 있는 도구를 만든다.

## 전체 아키텍처

```
[PDF 업로드] → [PDF 파싱 모듈] → [데이터 정제 모듈] → [분석 모듈] → [Streamlit 대시보드]
                                                            ↓
                                                    [향후 LLM 모듈]
```

### 프로젝트 구조

```
dart-financial-analyzer/
├── app.py                  # Streamlit 메인 앱
├── parser/
│   ├── pdf_extractor.py    # PDF에서 표/텍스트 추출
│   └── data_cleaner.py     # 추출 데이터 정제/구조화
├── analyzer/
│   ├── financial.py        # 재무 비율 계산
│   └── summary.py          # 핵심 지표 요약
├── data/                   # 업로드된 PDF 및 파싱 결과 저장
├── requirements.txt
└── README.md
```

## PDF 파싱 전략

### 표 데이터 (재무제표)

- **pdfplumber** 사용
- 추출 대상:
  - 재무상태표 — 자산, 부채, 자본 구조
  - 손익계산서 — 매출, 영업이익, 당기순이익
  - 현금흐름표 — 영업/투자/재무 활동 현금흐름
- 추출된 표 → pandas DataFrame → CSV 캐싱

### 텍스트 데이터 (사업 개요, 리스크)

- pdfplumber `extract_text()`로 추출
- 섹션 제목 키워드 매칭으로 구분 (예: "II. 사업의 내용", "III. 재무에 관한 사항")
- 1차에서는 텍스트 원문 표시, 향후 LLM 요약으로 확장

### 한계와 대응

- DART PDF 포맷이 기업마다 다름 → 100% 자동 파싱 어려움
- 테이블 추출 실패 시 사용자에게 안내
- 대기업 사업보고서 기준으로 파싱 로직 구축 후 점진적 확대

## 학습 중심 재무 분석

### 3단계 학습 흐름

**Step 1. 재무제표 읽기**
- 재무상태표, 손익계산서, 현금흐름표가 무엇인지 해설
- 추출된 실제 수치 위에 항목별 툴팁/해설 표시

**Step 2. 핵심 5대 지표**

| 지표 | 계산법 | 의미 |
|------|--------|------|
| 매출액 | 손익계산서 상단 | 회사의 외형 규모 |
| 영업이익률 | (영업이익 / 매출) × 100 | 본업의 수익성 |
| 부채비율 | (부채총계 / 자본총계) × 100 | 재무 건전성 |
| 유동비율 | (유동자산 / 유동부채) × 100 | 단기 지급 능력 |
| 영업활동 현금흐름 | 현금흐름표 | 본업에서 현금 창출 여부 |

각 지표마다 3가지 해설 제공:
- **이게 뭔가요?** — 한 줄 정의
- **왜 중요한가요?** — 투자심사 관점에서의 의미
- **이 회사는?** — 실제 수치 + 간단한 판단 기준

**Step 3. 심화 지표 (향후 확장)**
- ROE, ROA, 이자보상배율, 매출/영업이익 성장률
- 기업 비교 기능

### 대시보드 레이아웃

```
┌─────────────────────────────────────────┐
│  PDF 업로드                              │
├─────────────────────────────────────────┤
│  Tab 1: 재무제표 원문 (해설 포함)         │
│  Tab 2: 핵심 5대 지표 (학습 카드 형태)    │
│  Tab 3: 현금흐름 요약                    │
└─────────────────────────────────────────┘
```
